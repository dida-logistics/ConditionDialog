# ConditionDialog
条件选择器

# 如何引用： 
>1.在工程的build.gradle文件中加入：
<pre>
<code>
  allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}
</code>
</pre>
>2.在modle的build.gradle文件中加入：
<pre>
<code>
  dependencies {
	        compile 'com.github.dida-logistics:ConditionDialog:1.0.4'
	}
</code>
</pre>
